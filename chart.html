RESTful Routes

A table of all 7 RESTful routes

Name Path HTTP Verb Purpose Mongoose Method
Index /tutors GET List all tutors Tutor.find()
New /tutors/new GET Show new tutor form N/A
Create /tutors POST Create a new tutor, then redirect somewhere Tutor.create()
Show /tutors/:id GET Show info about one specific tutor Tutor.findById()
Edit /tutors/:id/edit GET Show edit form for one tutor Tutor.findById()
Update /tutors/:id PUT Update particular tutor, then redirect somewhere Tutor.findByIdAndUpdate()
Destroy /tutors/:id DELETE Delete a particular tutor, then redirect somewhere Tutor.findByIdAndRemove()
